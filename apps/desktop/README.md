# desktop

`apps/desktop` 是桌宠命令行助手的 Electron 桌面端。

## 作用

- 创建透明、置顶、无边框的桌宠窗口。
- 连接本地桥接服务，接收状态事件。
- 根据标准状态驱动角色展示、通知打断和详情面板。

## 目录结构

- `src/main/`：Electron 主进程入口，负责窗口生命周期和桌面壳能力。
- `src/preload/`：预加载层，预留主进程与渲染层的安全桥接入口。
- `src/renderer/`：渲染层界面，当前包含桌宠外壳和桥接客户端。

## 当前实现

- 主进程入口：`src/main/index.ts`
- 渲染层入口：`src/renderer/main.tsx`
- 桌宠壳组件：`src/renderer/pet-shell.tsx`
- 桥接客户端：`src/renderer/bridge-client.ts`

当前已经能启动桌面窗口并连接桥接服务，但界面仍处于原型阶段。
